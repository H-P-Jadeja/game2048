# 🎮 2048 Game (React)

A simple and interactive implementation of the classic **2048 puzzle game** built using **React.js**. The goal is to combine tiles with the same number to reach **2048**.

---

## 🚀 Demo

http://game2048-ten.vercel.app/

---

## 🧠 About the Game

2048 is a popular single-player sliding tile puzzle game where:

* You move tiles in **four directions** (up, down, left, right)
* Tiles with the same number **merge into one**
* The objective is to reach the **2048 tile**

## 🧑‍💻 Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/H-P-Jadeja/game2048.git
```

2. Navigate to the project folder:

```bash
cd game2048
```

3. Install dependencies:

```bash
npm install
```

4. Run the app:

```bash
npm start
```

---

## 🎯 How to Play

* Use **Arrow Keys** to move tiles
* Tiles merge when they have the same number
* Try to reach **2048** 🎉
* Game ends when no moves are left
